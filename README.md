# Mediator_z3pytranslation

## 概述
本仓库在[mediator的release版本](https://github.com/mediator-team/mediator.git)的基础上扩建形成，目的是实现从mediator到z3py的转化。

## 新增内容简要说明
- chat-record：github copilot配置文件（在vscode中通过import可查看已有对话记录）
- decompiled：反编译的org文件夹，涵盖了mediator语法解析与arduino generator等功能的主要代码，是完成项目需要学习的主要部分（？）
- example：copilot生成的简单例子，.bin是编译后的example
- gradle相关：一些花里胡哨的意外产物，无需在意
- MEDIATOR-LEARNING.md：copilot协助生成的帮助文档
